Get Started with Vulkron

Vulkron is a modern C++ renderer that combines Vulkan's performance with SDL3's cross-platform capabilities. Perfect for graphics research, engine development, and high-performance applications.

Quick Installation

# Clone the repository
git clone https://github.com/bisqottii/Vulkron.git
cd Vulkron

# Setup development environment
python build-scripts/conda.py setup
conda activate vk-env

# Build and install
python build-scripts/build.py --config release
python build-scripts/install.py

API Preview

#include <vulkron/gpu/device.hpp>
#include <vulkron/ui/window.hpp>

int main() {
    // Create window
    vulkron::ui::Window window(800, 600, "Vulkron App");
    
    // Initialize GPU
    vulkron::gpu::Device device;
    device.initialize();
    
    // Main loop
    while (window.isOpen()) {
        window.pollEvents();
        device.renderFrame();
    }
    
    return 0;
}
